# 参与贡献

欢迎参与 Capability Concentric 的开发！

- 提交 Issue 前，请先搜索是否已存在相同问题
- 提交 PR 前，请确保：
  - 通过 `npm run build` 无错误
  - 代码风格符合 ESLint 规则
  - UI 变更附上截图或简要说明

## 本地开发

1. 安装依赖：`npm i`
2. 启动开发：`npm run dev`
3. 构建产物：`npm run build`

## 提交规范

- feat: 新功能
- fix: 修复缺陷
- docs: 文档更新
- style: 格式变更（不影响逻辑）
- refactor: 重构
- perf: 性能优化
- test: 测试相关
- chore: 其他杂项
